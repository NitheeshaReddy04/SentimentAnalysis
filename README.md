# Sentiment Analysis Project

This project predicts whether a review is Positive or Negative using Machine Learning and NLP.

## Features
- Text preprocessing
- Sentiment prediction
- Logistic Regression model
- User input prediction

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP

## Files

- `sentiment_analysis.py` → Main Python code
- `reviews.csv` → Dataset
- `requirements.txt` → Libraries required
